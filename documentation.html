翻譯

雖然The Swift Programming Language 的 ePub 檔是用英文寫的，我們鼓勵你加它翻譯成其它語言。最好的做法是參與現有的翻譯或開始一個新的(有必要的話)。一個很好的例子是GitHub上的簡體中文翻譯：

The Swift Programming Language (簡體中文翻譯, GitHub project)

如果你知道一個高品質的翻譯正在進行中，請在swift-dev mailing list 上通知我們，我們會盡力在這此頁提供到該翻譯的連結。 繁體中文翻譯: https://github.com/tommy60703/the-swift-programming-language-in-traditional-chinese

API設計指南

Swift 3.0版 的目標之一是提供一個撰寫 Swift 程式的API設計指南。目前的API設計指南草案 包含命名，以及使用各種功能的方針。
